# Лабораторная работа №12. Жизненный цикл Activity (Activity Lifecycle)

## Grade Clicker
**Grade Clicker** — это интерактивное приложение-кликер для Android. Пользователь набирает баллы, кликая по изображению, а приложение демонстрирует работу жизненного цикла Activity и сохранение состояния при изменении конфигурации устройства.

## 2. Цель работы
Изучить жизненный цикл Activity в Android и научиться правильно обрабатывать переходы между состояниями. В ходе работы реализовано логирование callback-методов и решена проблема потери данных при повороте экрана.

## 3. Ответы на контрольные вопросы

*   **Какие три метода вызываются при запуске приложения?**  
    При запуске последовательно вызываются: `onCreate()`, `onStart()` и `onResume()`.
*   **Чем отличается onPause() от onStop()?**  
    `onPause()` вызывается, когда Activity частично видна (например, поверх открыт диалог) и теряет фокус. `onStop()` вызывается, когда Activity становится полностью невидимой для пользователя.
*   **Когда вызывается onRestart()?**  
    `onRestart()` вызывается только при возврате к Activity из состояния остановки (Stop), то есть когда приложение снова выходит на передний план после того, как было скрыто.
*   **Что происходит при повороте экрана?**  
    Происходит изменение конфигурации (configuration change): Android полностью уничтожает Activity (`onDestroy()`) и создает её заново (`onCreate()`), что приводит к потере несохраненных данных.
*   **В чём разница между remember и rememberSaveable?**  
    `remember` сохраняет значение только во время рекомпозиции. `rememberSaveable` сохраняет данные и при рекомпозиции, и при пересоздании Activity (например, при повороте экрана или смене языка).
*   **Почему onCreate() должен вызывать super.onCreate()?**  
    Это необходимо для выполнения базовой инициализации Activity на уровне операционной системы, без которой приложение не сможет корректно запуститься.
*   **Для чего используется класс Log?**  
    Класс `Log` используется для записи отладочных сообщений в консоль Logcat, что позволяет разработчику отслеживать вызовы методов и состояние приложения в реальном времени.
*   **Какой метод вызывается только один раз за время жизни Activity?**  
    Метод `onCreate()` вызывается один раз при создании экземпляра Activity.
